# NibiruChain/gitbook-docs


## ⚙️ — Quick Start: Documentation and Demo

#### Installation for development on localhost

```sh
nvm use 12.22.12
npm install -g yarn

yarn install
yarn run build
yarn lint && yarn test

yarn honkit:build
yarn honkit:serve
```

The last output line should say "Serving book on http://localhost:4000".
